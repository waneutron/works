Import Rekod dari Excel

Muat naik fail Excel (.xlsx atau .xls) untuk mengimport rekod dalam jumlah besar.

Format Fail Excel

Pastikan fail Excel anda mengikuti format ini:

  • row A: Tajuk
  • row B: Nama Fail (Author)
  • row C: Rujukan (ISBN)
  • row D: Bilangan (Quantity)
  • row E: Tingkat (Room)
  • row F: Almari (Shelf)
  • row G: Bahagian (Section)
Saiz maksimum: 16 MB
Kembali

Tip: Anda boleh membuat fail template Excel dengan maklumat dalam format di atas.
